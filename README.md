# MikroTik Manager

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Flask](https://img.shields.io/badge/Flask-3.0-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

**MikroTik Manager** — это веб-приложение для централизованного управления устройствами MikroTik RouterOS. Программа позволяет автоматизировать проверку обновлений, выполнение обновлений, создание резервных копий и управление множеством устройств через единый веб-интерфейс.

## Быстрый старт

### 1. Клонируйте репозиторий и перейдите в его директорию
```bash
git clone https://github.com/RegentsVoice/MikrotikManager.git
cd ./mikrotik-manager
```
### 2. Создайте виртуальное окружение
```bash
pyython -m venv venv
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Активируйте виртуальное окружение (Выберете для вашего shell)
Fish
```bash
source ./venv/bin/activate.fish
```
Bash
```bash
source ./venv/bin/activate
```

Powershell
```bash
. ./venv/bin/activate.ps1
```
### 5. Запустите приложение и следуйте подсказкав в консоли
```bash
python ./app.py
```
