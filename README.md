# Mikrotik-update-manager

Инструмент для централизованного обновления и резервного копирования
устройств **MikroTik**.

## Описание

**Mikrotik-update-manager** --- это простой инструмент для
автоматизации обслуживания сетевых устройств MikroTik.\
Он облегчает обновление прошивок, создаёт резервные копии и позволяет
управлять несколькими устройствами централизованно.

## Возможности

- Автоматическое обновление прошивок MikroTik\
- Создание резервных копий конфигураций перед обновлением и по расписанию\
- Группировка и централизованное управление устройствами\
- Подробное логирование всех операций\
- Гибкая настройка через конфигурационные файлы

## Установка

``` bash
git clone https://github.com/RegentsVoice/Mikrotik-update-manager.git
cd Mikrotik-update-manager
pip install -r requirements.txt
```

## Настройка

В каталоге **config/** находятся файлы конфигурации. Вам необходимо
указать:

-   список устройств (IP, порт, логин/пароль или ключ)\
-   параметры резервного копирования\
-   параметры обновления RouterOS

Например:

``` yaml
devices:
  - name: router
    host: 192.168.1.1
    username: admin
    password: secret
    group: core
```

## Запуск

Старт программы:

``` bash
python main.py
```


## Логи

Все события записываются в каталог **logs/**.
